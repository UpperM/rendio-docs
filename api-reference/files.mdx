---
title: "GET /api/v1/files/{id}"
description: "Retrieve metadata for a previously rendered file by its ID."
api: "GET https://rendio.dev/api/v1/files/{id}"
---

Returns the CDN URL and metadata for a previously rendered file. The file
must belong to the authenticated account.

For **private files**, this endpoint generates a fresh signed URL on every
call — use it to refresh expired signed URLs without re-rendering.

## Authentication

Bearer token. See [Authentication](/authentication).

## Path parameters

<ParamField path="id" type="string" required>
  File identifier — format `rd_[a-f0-9]{16}` (or legacy 32-hex-char format).
</ParamField>

## Query parameters

<ParamField query="ttl" type="integer">
  Signed URL TTL in seconds. Range `60` – `604800` (7 days). Only applies to
  private files. Defaults to `3600` (1 hour). Ignored for public files.
</ParamField>

## Response

`200 OK`, `Content-Type: application/json`

```json
{
  "id": "rd_abc1234567890def",
  "url": "https://cdn.rendio.dev/s/rd_abc1234567890def.pdf?expires=1745364000&sig=a1b2c3...",
  "pages": 2,
  "size": 142857,
  "expiresAt": "2026-05-21T12:00:00.000Z",
  "urlExpiresAt": "2026-04-21T13:00:00.000Z"
}
```

<ResponseField name="id" type="string">File identifier.</ResponseField>
<ResponseField name="url" type="string">CDN download URL. Signed with expiry for private files.</ResponseField>
<ResponseField name="pages" type="integer">Page count (PDF only; 0 for screenshots).</ResponseField>
<ResponseField name="size" type="integer">File size in bytes.</ResponseField>
<ResponseField name="expiresAt" type="string | null">
  ISO&nbsp;8601 timestamp when the file is deleted from storage.
</ResponseField>
<ResponseField name="urlExpiresAt" type="string | null">
  ISO&nbsp;8601 timestamp when the signed URL expires. `null` for public files.
  Call this endpoint again to get a fresh URL.
</ResponseField>

## Errors

| Status | Code | Cause |
|---|---|---|
| 400 | `validation_error` | `id` does not match the expected format, or `ttl` is out of range |
| 401 | `invalid_api_key` | Missing, malformed, or revoked key |
| 404 | `not_found` | File does not exist, is expired, belongs to a different account, or was rendered in preview mode |

## Examples

<RequestExample>

```bash cURL
# Public file
curl https://rendio.dev/api/v1/files/rd_abc1234567890def \
  -H "Authorization: Bearer $RENDIO_API_KEY"

# Private file — refresh signed URL with 5-minute TTL
curl "https://rendio.dev/api/v1/files/rd_abc1234567890def?ttl=300" \
  -H "Authorization: Bearer $RENDIO_API_KEY"
```

```ts Node
// Public file
const pdf = await rendio.pdfs.get("rd_abc1234567890def");
console.log(pdf.url);

// Private file — refresh with 5-minute TTL
const fresh = await rendio.pdfs.get("rd_abc1234567890def", { ttl: 300 });
console.log(fresh.url);           // signed URL
console.log(fresh.urlExpiresAt);  // when it expires
```

```python Python
# Public file
pdf = client.pdfs.get("rd_abc1234567890def")
print(pdf.url)

# Private file — refresh with 5-minute TTL
fresh = client.pdfs.get("rd_abc1234567890def", ttl=300)
print(fresh.url)             # signed URL
print(fresh.url_expires_at)  # when it expires
```

```go Go
// Public file
pdf, err := client.Pdfs.Get(ctx, "rd_abc1234567890def", nil)
if err != nil { return err }
fmt.Println(pdf.URL)

// Private file — refresh with 5-minute TTL
fresh, err := client.Pdfs.Get(ctx, "rd_abc1234567890def", &rendio.PdfGetOptions{TTL: rendio.Int(300)})
if err != nil { return err }
fmt.Println(fresh.URL)          // signed URL
fmt.Println(*fresh.URLExpiresAt) // when it expires
```

</RequestExample>

## Retention

Files are kept per plan:

| Plan | Retention |
|---|---|
| Free | 24 hours |
| Starter | 30 days |
| Pro | 30 days |

Past the window, the `id` returns `404` and the CDN URL returns `404` too.
Store the `url` at render time, or re-render when you need the file.

Preview-mode renders (`preview: true`) are never indexed by this endpoint —
they are transient and not tied to an account.
