Method-level reference for the `pdfs` resource. Parameter semantics are
shared with [`POST /api/v1/render`](/api-reference/render) — this page focuses
on SDK ergonomics.

## `pdfs.create(params, options?)`

Render HTML (or a URL) to PDF / PNG / JPEG. Exactly one of `html` or
`source` must be provided.

<Tabs>
  <Tab title="Node">
    ```ts
    const pdf = await rendio.pdfs.create(
      {
        html: "<h1>Invoice #{{n}}</h1>",
        data: { n: "2026-042" },
        pageSize: "A4",
      },
      { idempotencyKey: "invoice-8F27" } // optional per-request overrides
    );
    ```
  </Tab>
  <Tab title="Python">
    ```python
    pdf = client.pdfs.create(
        html="<h1>Invoice #{{n}}</h1>",
        data={"n": "2026-042"},
        page_size="A4",
        idempotency_key="invoice-8F27",
    )
    ```
  </Tab>
  <Tab title="Go">
    ```go
    pdf, err := client.Pdfs.Create(ctx,
        rendio.PdfCreateParams{
            HTML:     "<h1>Invoice #{{n}}</h1>",
            Data:     map[string]any{"n": "2026-042"},
            PageSize: rendio.PageSizeA4,
        },
        rendio.WithIdempotencyKey("invoice-8F27"),
    )
    ```
  </Tab>
</Tabs>

### URL mode (default)

`response: "url"` uploads the rendered file to Rendio's CDN and returns
metadata + a CDN URL.

```ts
const pdf = await rendio.pdfs.create({
  html: "<h1>Invoice</h1>",
  pageSize: "A4",
});

pdf.id;            // "rd_abc1234567890def"
pdf.url;           // "https://cdn.rendio.dev/f/rd_...pdf"
pdf.pages;         // 1
pdf.size;          // bytes
pdf.expiresAt;     // ISO 8601 — when the file is deleted from storage
pdf.urlExpiresAt;  // ISO 8601 — when the signed URL expires (null for public)
pdf.meta.chromiumVersion;
pdf.meta.renderTimeMs;
```

### Buffer mode

`response: "buffer"` streams raw bytes back — no CDN storage, no storage
quota consumed.

<Tabs>
  <Tab title="Node">
    ```ts
    import { writeFileSync } from "node:fs";

    const result = await rendio.pdfs.create({
      html: "<h1>Invoice</h1>",
      response: "buffer",
    });

    writeFileSync("invoice.pdf", Buffer.from(result.buffer));
    ```
  </Tab>
  <Tab title="Python">
    ```python
    result = client.pdfs.create(html="<h1>Invoice</h1>", response="buffer")

    with open("invoice.pdf", "wb") as f:
        f.write(result.buffer)
    ```
  </Tab>
  <Tab title="Go">
    ```go
    result, err := client.Pdfs.CreateBuffer(ctx, rendio.PdfCreateParams{
        HTML: "<h1>Invoice</h1>",
    })
    if err != nil { log.Fatal(err) }

    os.WriteFile("invoice.pdf", result.Buffer, 0644)
    ```

    Go uses a separate `CreateBuffer` method (not a parameter on `Create`).
    `CreateBufferWithResponse` returns both the result and the raw `*http.Response`.
  </Tab>
</Tabs>

### Per-request options

| Option | Type | Purpose | SDKs |
|---|---|---|---|
| `timeout` | number (ms) | Override the client-level timeout | All |
| `maxRetries` | number | Override the client-level retry count | All |
| `signal` | `AbortSignal` | Cancel the request from the caller | Node only |
| `idempotencyKey` | string | Deduplicate retries (URL mode only) | All |

Go uses `context.Context` (first param) for cancellation instead of `signal`.
The Python SDK does not currently support request cancellation.

## `pdfs.get(id)`

Fetch metadata for a previously rendered file.

<Tabs>
  <Tab title="Node">
    ```ts
    const pdf = await rendio.pdfs.get("rd_abc1234567890def");
    console.log(pdf.url);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    pdf = client.pdfs.get("rd_abc1234567890def")
    print(pdf.url)
    ```
  </Tab>
  <Tab title="Go">
    ```go
    pdf, err := client.Pdfs.Get(ctx, "rd_abc1234567890def")
    if err != nil { return err }
    fmt.Println(pdf.URL)
    ```
  </Tab>
</Tabs>

Throws `NotFoundError` when the ID doesn't exist, has expired, or was
rendered in preview mode (preview files are not indexed).

## `pdfs.merge(params)`

Merge two or more PDFs into a single file. Sources must be file IDs from
previous renders — URL sources are not yet supported. **Starter plan or above.**

<Tabs>
  <Tab title="Node">
    ```ts
    const merged = await rendio.pdfs.merge({
      sources: [{ id: cover.id }, { id: report.id }],
      filename: "full-report",
    });
    console.log(merged.url);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    merged = client.pdfs.merge(
        sources=[{"id": cover.id}, {"id": report.id}],
        filename="full-report",
    )
    print(merged.url)
    ```
  </Tab>
  <Tab title="Go">
    ```go
    merged, err := client.Pdfs.Merge(ctx, rendio.PdfMergeParams{
        Sources:  []rendio.PdfSource{{ID: cover.ID}, {ID: report.ID}},
        Filename: rendio.String("full-report"),
    })
    if err != nil { return err }
    fmt.Println(merged.URL)
    ```
  </Tab>
</Tabs>

Returns the same shape as `pdfs.create()` in URL mode.
See [`POST /api/v1/pdfs/merge`](/api-reference/merge) for the full parameter
reference, plan limits, and error codes.

## `pdfs.extract(params)`

Extract a subset of pages from an existing PDF. **Starter plan or above.**

<Tabs>
  <Tab title="Node">
    ```ts
    const extracted = await rendio.pdfs.extract({
      source: { id: "rd_abc1234567890def" },
      pageRanges: "1-3",
      filename: "summary",
    });
    console.log(extracted.url);
    ```
  </Tab>
  <Tab title="Python">
    ```python
    extracted = client.pdfs.extract(
        source={"id": "rd_abc1234567890def"},
        page_ranges="1-3",
        filename="summary",
    )
    print(extracted.url)
    ```
  </Tab>
  <Tab title="Go">
    ```go
    extracted, err := client.Pdfs.Extract(ctx, rendio.PdfExtractParams{
        Source:     rendio.PdfSource{ID: "rd_abc1234567890def"},
        PageRanges: "1-3",
        Filename:   rendio.String("summary"),
    })
    if err != nil { return err }
    fmt.Println(extracted.URL)
    ```
  </Tab>
</Tabs>

`pageRanges` uses Chromium's 1-indexed format: `"1-3"`, `"1, 3, 5-7"`, `"2-"`.
Returns the same shape as `pdfs.create()` in URL mode.
See [`POST /api/v1/pdfs/extract`](/api-reference/extract) for the full parameter
reference and error codes.

## Post-processing with `pdfs.create()`

The `post` option applies watermarking, password protection, and linearization
after rendering. It is also accepted by `merge` and `extract`.

<Tabs>
  <Tab title="Node">
    ```ts
    const pdf = await rendio.pdfs.create({
      html: "<h1>Confidential</h1>",
      post: {
        watermark: {
          text: "DRAFT",
          opacity: 0.2,
          rotation: -45,
          repeat: true,
        },
        protection: {
          ownerPassword: "owner-secret",
          permissions: { print: true, copy: false, modify: false },
        },
      },
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    pdf = client.pdfs.create(
        html="<h1>Confidential</h1>",
        post={
            "watermark": {
                "text": "DRAFT",
                "opacity": 0.2,
                "rotation": -45,
                "repeat": True,
            },
            "protection": {
                "owner_password": "owner-secret",
                "permissions": {"print": True, "copy": False, "modify": False},
            },
        },
    )
    ```
  </Tab>
  <Tab title="Go">
    ```go
    pdf, err := client.Pdfs.Create(ctx, rendio.PdfCreateParams{
        HTML: "<h1>Confidential</h1>",
        Post: &rendio.PostOptions{
            Watermark: &rendio.WatermarkOptions{
                Text:     rendio.String("DRAFT"),
                Opacity:  rendio.Float64(0.2),
                Rotation: rendio.Float64(-45),
                Repeat:   rendio.Bool(true),
            },
            Protection: &rendio.ProtectionOptions{
                OwnerPassword: rendio.String("owner-secret"),
                Permissions: &rendio.PermissionOptions{
                    Print:  rendio.Bool(true),
                    Copy:   rendio.Bool(false),
                    Modify: rendio.Bool(false),
                },
            },
        },
    })
    ```
  </Tab>
</Tabs>

## Return types

### Node / TypeScript

```ts
import type {
  PdfCreateParams,
  PdfMergeParams,
  PdfExtractParams,
  PdfSource,
  PdfUrlResult,
  PdfBufferResult,
  RenderMeta,
  PostOptions,
  WatermarkOptions,
  ProtectionOptions,
  PermissionOptions,
  RendioOptions,
  PageSize,
  ResponseMode,
} from "rendio";
```

| Type | Shape |
|---|---|
| `PdfUrlResult` | `{ id, url, pages, size, expiresAt, urlExpiresAt, meta }` |
| `PdfBufferResult` | `{ buffer: ArrayBuffer, meta }` |
| `RenderMeta` | `{ chromiumVersion?, renderTimeMs?, phases?, rateLimit? }` |
| `PdfMergeParams` | `{ sources: PdfSource[], filename?, storage?, post? }` |
| `PdfExtractParams` | `{ source: PdfSource, pageRanges: string, filename?, storage?, post? }` |
| `PdfSource` | `{ id: string } \| { url: string }` |
| `PostOptions` | `{ watermark?, protection?, linearize? }` |
| `WatermarkOptions` | `{ text?, image?, opacity?, rotation?, position?, repeat?, fontSize?, color? }` |
| `ProtectionOptions` | `{ password?, ownerPassword?, permissions? }` |
| `PermissionOptions` | `{ print?, copy?, modify?, annotate? }` |

#### `meta.rateLimit`

When rate limit headers are present in the response, `meta.rateLimit` is
populated:

| Field | Type | Description |
|---|---|---|
| `limit` | `number` | Max requests per window |
| `remaining` | `number` | Requests remaining |
| `resetAt` | `number` | Unix timestamp when the window resets |

### Python

| Type | Fields |
|---|---|
| `PdfCreateUrlResult` | `id`, `url`, `pages`, `size`, `expires_at`, `url_expires_at`, `meta` |
| `PdfBufferResult` | `buffer` (bytes), `meta` |
| `RenderMeta` | `chromium_version?`, `render_time_ms?`, `phases?`, `rate_limit?` |
| `PdfMergeParams` | `sources`, `filename?`, `storage?`, `post?` |

## Python async client

The Python SDK ships both sync and async clients. Use `AsyncRendio` for
FastAPI, async task queues, or any `asyncio`-based application.

```python
import rendio

# Sync client (Django, Flask, scripts)
client = rendio.Rendio()
pdf = client.pdfs.create(html="<h1>Invoice</h1>")

# Async client (FastAPI, Celery async tasks)
async_client = rendio.AsyncRendio()
pdf = await async_client.pdfs.create(html="<h1>Invoice</h1>")
```

The async client uses `httpx.AsyncClient` under the hood. Use it as a
context manager for proper connection cleanup:

```python
async with rendio.AsyncRendio() as client:
    pdf = await client.pdfs.create(html="<h1>Invoice</h1>")
    print(pdf.url)
```

### FastAPI example

Use FastAPI's `lifespan` protocol to properly manage the client connection
pool. Creating `AsyncRendio()` at module scope without cleanup leaks the
underlying `httpx.AsyncClient`.

```python
from contextlib import asynccontextmanager
from fastapi import FastAPI
from fastapi.responses import Response
from pydantic import BaseModel
import rendio

@asynccontextmanager
async def lifespan(app: FastAPI):
    app.state.rendio = rendio.AsyncRendio()
    yield
    await app.state.rendio.aclose()

app = FastAPI(lifespan=lifespan)

class PdfRequest(BaseModel):
    html: str
    page_size: str = "A4"

@app.post("/api/pdf")
async def generate_pdf(req: PdfRequest):
    client: rendio.AsyncRendio = req.app.state.rendio
    try:
        result = await client.pdfs.create(
            html=req.html, page_size=req.page_size, response="buffer",
        )
        return Response(
            content=result.buffer,
            media_type="application/pdf",
            headers={"Content-Disposition": 'attachment; filename="document.pdf"'},
        )
    except rendio.AuthenticationError:
        return Response(status_code=500, content="PDF service unavailable")
    except rendio.QuotaExceededError:
        return Response(status_code=503, content="Quota exhausted")
    except rendio.RendioError as e:
        return Response(status_code=e.status_code or 500, content=e.message)
```

## Go additional methods

The Go SDK exposes additional variants for advanced use cases:

| Method | Returns | Use case |
|---|---|---|
| `Pdfs.Create(ctx, params, opts...)` | `*PdfCreateURLResult, error` | URL mode (default) |
| `Pdfs.CreateBuffer(ctx, params, opts...)` | `*PdfBufferResult, error` | Buffer mode |
| `Pdfs.CreateWithResponse(ctx, params, opts...)` | `*PdfCreateURLResult, *http.Response, error` | URL mode + raw HTTP response |
| `Pdfs.CreateBufferWithResponse(ctx, params, opts...)` | `*PdfBufferResult, *http.Response, error` | Buffer mode + raw HTTP response |
| `Pdfs.Get(ctx, id, opts...)` | `*PdfURLResult, error` | Fetch file metadata |
| `Pdfs.GetWithResponse(ctx, id, opts...)` | `*PdfURLResult, *http.Response, error` | Metadata + raw HTTP response |
| `Pdfs.Merge(ctx, params, opts...)` | `*PdfCreateURLResult, error` | Merge PDFs |
| `Pdfs.MergeWithResponse(ctx, params, opts...)` | `*PdfCreateURLResult, *http.Response, error` | Merge + raw HTTP response |
| `Pdfs.Extract(ctx, params, opts...)` | `*PdfCreateURLResult, error` | Extract pages |
| `Pdfs.ExtractWithResponse(ctx, params, opts...)` | `*PdfCreateURLResult, *http.Response, error` | Extract + raw HTTP response |

## Testing

Point the SDK at a local mock server using `WithBaseURL` (or its equivalent)
so tests never hit the real API.

<Tabs>
  <Tab title="Node">
    Use [msw](https://mswjs.io/) or a lightweight HTTP server. Pass the
    server's URL via the `baseUrl` constructor option.

    ```ts
    import { http, HttpResponse } from "msw";
    import { setupServer } from "msw/node";
    import Rendio from "rendio";
    import { afterAll, beforeAll, expect, test } from "vitest";

    const mockPdf = {
      id: "rd_test123",
      url: "https://cdn.rendio.dev/f/rd_test123.pdf",
      pages: 1,
      size: 1024,
      expiresAt: null,
      meta: {},
    };

    const server = setupServer(
      http.post("http://localhost:9876/api/v1/render", () =>
        HttpResponse.json(mockPdf),
      ),
    );

    beforeAll(() => server.listen());
    afterAll(() => server.close());

    test("creates a PDF", async () => {
      const rendio = new Rendio("rk_test_xxx", {
        baseUrl: "http://localhost:9876",
        maxRetries: 0,
      });

      const pdf = await rendio.pdfs.create({ html: "<h1>Test</h1>" });
      expect(pdf.id).toBe("rd_test123");
      expect(pdf.url).toContain("rd_test123");
    });
    ```
  </Tab>
  <Tab title="Python">
    Use [respx](https://lundberg.github.io/respx/) to intercept `httpx`
    requests. Point the client at a test base URL so the mock matcher is
    stable.

    ```python
    import respx
    from httpx import Response
    import rendio

    MOCK_PDF = {
        "id": "rd_test123",
        "url": "https://cdn.rendio.dev/f/rd_test123.pdf",
        "pages": 1,
        "size": 1024,
        "expiresAt": None,
        "meta": {},
    }

    @respx.mock
    def test_create_pdf():
        respx.post("http://test/api/v1/render").mock(
            return_value=Response(200, json=MOCK_PDF)
        )

        client = rendio.Rendio(
            api_key="rk_test_xxx",
            base_url="http://test",
            max_retries=0,
        )

        pdf = client.pdfs.create(html="<h1>Test</h1>")
        assert pdf.id == "rd_test123"
        assert "rd_test123" in pdf.url
    ```
  </Tab>
  <Tab title="Go">
    Use `net/http/httptest` to spin up a local server, then pass its URL
    via `rendio.WithBaseURL`.

    ```go
    package pdf_test

    import (
        "context"
        "encoding/json"
        "net/http"
        "net/http/httptest"
        "testing"

        "github.com/rendio/rendio-go"
    )

    func TestCreatePdf(t *testing.T) {
        mockPdf := map[string]any{
            "id":    "rd_test123",
            "url":   "https://cdn.rendio.dev/f/rd_test123.pdf",
            "pages": 1,
            "size":  1024,
        }

        server := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
            if r.URL.Path != "/api/v1/render" {
                t.Errorf("unexpected path: %s", r.URL.Path)
            }
            w.Header().Set("Content-Type", "application/json")
            json.NewEncoder(w).Encode(mockPdf)
        }))
        defer server.Close()

        client, err := rendio.NewClient(
            rendio.WithAPIKey("rk_test_xxx"),
            rendio.WithBaseURL(server.URL),
            rendio.WithMaxRetries(0),
        )
        if err != nil {
            t.Fatal(err)
        }

        pdf, err := client.Pdfs.Create(context.Background(), rendio.PdfCreateParams{
            HTML: "<h1>Test</h1>",
        })
        if err != nil {
            t.Fatal(err)
        }

        if pdf.ID != "rd_test123" {
            t.Errorf("got id %q, want rd_test123", pdf.ID)
        }
    }
    ```
  </Tab>
</Tabs>

<Tip>
  Set `maxRetries: 0` in tests to avoid backoff delays and make failures
  immediate. See [Retries & timeouts](/sdks/retries-and-timeouts) for more.
</Tip>
