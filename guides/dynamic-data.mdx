Rendio uses [Handlebars](https://handlebarsjs.com/) for templating. Pass
values through `data`, reference them with `{{variable}}`.

## Quick example

<Tabs>
  <Tab title="Node">
    ```ts
    const pdf = await rendio.pdfs.create({
      html: `
        <h1>Invoice #{{invoice.number}}</h1>
        <p>Client: {{client.name}}</p>
        <p>Total: {{invoice.total}}</p>
      `,
      data: {
        invoice: { number: "2026-042", total: "$4,320.00" },
        client: { name: "Acme Corp" },
      },
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    pdf = client.pdfs.create(
        html="""
            <h1>Invoice #{{invoice.number}}</h1>
            <p>Client: {{client.name}}</p>
            <p>Total: {{invoice.total}}</p>
        """,
        data={
            "invoice": {"number": "2026-042", "total": "$4,320.00"},
            "client": {"name": "Acme Corp"},
        },
    )
    ```
  </Tab>
  <Tab title="Go">
    ```go
    pdf, err := client.Pdfs.Create(ctx, rendio.PdfCreateParams{
        HTML: `
            <h1>Invoice #{{invoice.number}}</h1>
            <p>Client: {{client.name}}</p>
            <p>Total: {{invoice.total}}</p>
        `,
        Data: map[string]any{
            "invoice": map[string]any{"number": "2026-042", "total": "$4,320.00"},
            "client":  map[string]any{"name": "Acme Corp"},
        },
    })
    ```
  </Tab>
</Tabs>

## Variables

Dot notation for nested objects:

```html
<p>Company: {{company.name}}</p>
<p>Contact: {{company.contact.email}}</p>
```

### HTML escaping

Handlebars escapes `<`, `>`, `&`, `"`, `'` by default. Triple braces
`{{{variable}}}` output raw HTML.

<Warning>
  Only use `{{{triple braces}}}` with trusted content. Never with
  user-provided input — it's an XSS vector.
</Warning>

## Loops — `{{#each}}`

```html
<tbody>
  {{#each items}}
  <tr>
    <td>{{@index}}.</td>
    <td>{{description}}</td>
    <td>{{qty}}</td>
    <td>{{total}}</td>
  </tr>
  {{/each}}
</tbody>
```

Loop context variables:

| Variable | Meaning |
|---|---|
| `@index` | Zero-based index |
| `@first` / `@last` | Boolean — first / last iteration |
| `@key` | Property name (when iterating an object) |
| `this` | Current item |

Empty-array fallback with `{{else}}`:

```html
{{#each items}}
  <tr><td>{{description}}</td></tr>
{{else}}
  <tr><td>No items.</td></tr>
{{/each}}
```

## Conditionals — `{{#if}}` / `{{#unless}}`

```html
{{#if client.vatNumber}}
  <p>VAT: {{client.vatNumber}}</p>
{{/if}}

{{#if invoice.isPaid}}
  <span class="badge paid">PAID</span>
{{else}}
  <span class="badge pending">PENDING</span>
{{/if}}

{{#unless invoice.isPaid}}
  <p class="warning">Payment due {{invoice.dueDate}}</p>
{{/unless}}
```

Handlebars treats these as falsy: `false`, `undefined`, `null`, `""`, `0`,
`[]`. Everything else is truthy.

## Formatting numbers and dates

Handlebars has no built-in formatters. Format in your app before
sending:

<Tabs>
  <Tab title="Node">
    ```ts
    const fmtCurrency = (n: number) =>
      new Intl.NumberFormat("en-US", { style: "currency", currency: "USD" }).format(n);

    const fmtDate = (d: Date) =>
      new Intl.DateTimeFormat("en-US", { dateStyle: "long" }).format(d);

    const pdf = await rendio.pdfs.create({
      html: "<p>Total: {{total}}</p><p>Date: {{date}}</p>",
      data: {
        total: fmtCurrency(4320),
        date: fmtDate(new Date()),
      },
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import locale
    from datetime import date

    locale.setlocale(locale.LC_ALL, "en_US.UTF-8")

    pdf = client.pdfs.create(
        html="<p>Total: {{total}}</p><p>Date: {{date}}</p>",
        data={
            "total": locale.currency(4320, grouping=True),
            "date": date.today().strftime("%B %d, %Y"),
        },
    )
    ```
  </Tab>
  <Tab title="Go">
    ```go
    import "fmt"
    import "time"

    pdf, err := client.Pdfs.Create(ctx, rendio.PdfCreateParams{
        HTML: "<p>Total: {{total}}</p><p>Date: {{date}}</p>",
        Data: map[string]any{
            "total": fmt.Sprintf("$%,.2f", 4320.0),
            "date":  time.Now().Format("January 02, 2006"),
        },
    })
    ```
  </Tab>
</Tabs>

## Syntax reference

| Syntax | Purpose |
|---|---|
| `{{name}}` | Output variable (HTML-escaped) |
| `{{{name}}}` | Output raw HTML |
| `{{obj.prop}}` | Nested property |
| `{{#each items}}...{{/each}}` | Array loop |
| `{{#if x}}...{{/if}}` | Conditional |
| `{{#if x}}...{{else}}...{{/if}}` | Conditional with else |
| `{{#unless x}}...{{/unless}}` | Negated conditional |
| `{{@index}}` | Loop index |
| `{{@first}}` / `{{@last}}` | First / last flags |
| `{{this}}` | Current iteration value |
| `{{!-- comment --}}` | Not rendered |

<Tip>
  Pre-render complex templates server-side before sending to Rendio. For
  large datasets (100+ items) consider paginating across multiple PDFs —
  the 1&nbsp;MB HTML limit is per request.
</Tip>
